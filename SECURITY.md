# 安全与隐私

## 支持范围

安全修复以当前主分支为准。项目处于早期阶段，暂不承诺旧版本长期维护。

## 报告问题

请优先使用代码托管平台的私密安全报告功能。若只能使用公开 Issue，请不要粘贴 API Key、内网地址、真实公文、个人信息或可直接利用的完整敏感细节。

## 文档数据

- 未启用 AI 时，OfficeTool 不会主动将文档内容发送到网络。
- 启用 AI 后，文档纯文本会发送到用户配置的服务地址，使用者应自行确认数据处理权限。
- GUI 保存的 AI Key 使用 Windows DPAPI 按当前用户加密；导出的通用配置不包含 Key。
- 发布 EXE 时必须从干净源码构建，不得打包 `%APPDATA%\OfficeTool`、`.env`、用户配置或测试文档。
- `.doc` 转换会禁用 Office 宏并只读打开源文件，但无法消除旧版 Office 解析器自身的安全风险，只应处理可信文件。
