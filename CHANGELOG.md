# 更新记录

## v0.1.1 - 2026-06-26

公开测试修复版本。

### 修复与优化

- 优化普通公文、红头文件、函和会议纪要的校对导出细节。
- 修正结尾日期、落款单位、版记落页和附件排版的若干边界情况。
- 优化红头和版记按需生成逻辑，避免重复生成并增强缺失结构处理。
- 改进 `.doc` 离线转换、中文错误提示和用户自定义配置交互。
- 清理公开发布范围，发布包不包含 API Key、测试文档或本机配置。

## v0.1.0 - 2026-06-24

首个公开测试版本。

### 主要功能

- 支持普通公文、普通红头文件、红头文件（函）和红头文件（会议纪要）。
- 支持普通红头附带制度正文。
- 支持 `.doc`、`.docx`、`.txt`、`.md` 和界面直接输入文稿。
- `.doc` 通过本机 Microsoft Word 或 WPS 完成离线转换。
- 支持页面、网格、字体、标题层级、附件、页码、落款、日期和版记整理。
- 支持按需生成完整缺失的红头和版记。
- 支持自定义配置持久化和可选 AI 文本校对。

### 注意事项

- 本版本为公开测试版，重要公文仍需人工最终复核。
- AI 默认关闭，发布包不包含 API Key。
- `.doc` 转换需要电脑已安装 Word 或支持 COM 自动化的 WPS。
- 程序未进行商业代码签名，Windows 首次运行时可能显示安全提示。
